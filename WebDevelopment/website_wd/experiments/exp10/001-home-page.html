Home Page



In this experiment, I designed the home page for my final project 'Moviebook'. The home page displays the movies playing in theatres right now using the Rotten Tomatoes API. The link for that page is:

  • http://project-rmcs5610.rhcloud.com/movieapp#/
The app is a single page app without any whole page refresh. Code snippet below: 

        <div>
        <h4 class="margin40">In theaters</h4>
        <div id="results">
        <table class="table">
        <thead>
        <tr>
        <th>Poster</th>
        <th>Movie Name</th>
        <th>Rating</th>
        <th>Year</th>
        <th>Critics</th>
        <th>Audience</th>
                    </tr>
                </thead>
        <tbody>
        <tr ng-repeat="m in movies">
        <td>
        <a href="#/movie/{{m.id}}">
        <img ng-src="{{m.posters.thumbnail}}" />
                            </a>
                        </td>
        <td>
        <a href="#/movie/{{m.id}}">{{m.title}}</a>
                        </td>
        <td>{{m.mpaa_rating}}</td>
        <td>{{m.year}}</td>
        <td>{{m.ratings.critics_score}} %</td>
        <td>{{m.ratings.audience_score}} %</td>
                    </tr>
                </tbody>
            </table>
        </div>
        <div id="message">
        </div>
        <div class="centerAlignMargin">
        <button class="btn btn-primary width80px" ng-click="previousPage()">Previous</button>
        <button class="btn btn-primary width80px" ng-click="nextPage()">Next</button>
        </div>
    </div>

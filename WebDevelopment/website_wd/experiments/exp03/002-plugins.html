jQuery Plugins Demo

Knob

jQuery Knob (github) is a plugin for turning input elements into touchable jQuery dials. It is built using canvas, and is fully customizable by setting data attributes on the inputs like show below: 

    <input type="text" value="75" class="dial">
    <script type="text/javascript" src="js/jquery.knob.js"></script>
        $(function () {
          $(".dial").knob();
        });
    </script>

backstretch

Backstretch is a simple jQuery plugin that allows you to add a dynamically-resized, slideshow-capable background image to any page or element. It will stretch any image to fit the page or block-level element, and will automatically resize as the window or element size changes. Images are fetched after your page is loaded, so users don’t have to wait for the (often large) image to download before using your site. 

    <script type="text/javascript" src="js/jquery.backstretch.min.js">
       $.backstretch("images/1280-boston-ma-smart-city.jpg");
    </script>

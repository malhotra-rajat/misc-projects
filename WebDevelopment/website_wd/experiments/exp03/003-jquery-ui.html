In this experiment, I have implemented the sortable widget of jQuery, which the user can interact with using drag-and-drop. The id attribute that is attached on the parent element as shown in the code snippet is used for activating the DOM element to jQuery UI. The default styles for the <li> elements consist of the background gradient colors, the length of the elements, the borders, the text styles. The border for the first and last child of <li> is specifically removed. 

        <ul id="sortable">
            <li class="ui-state-default">Item 1</li>
            <li class="ui-state-default">Item 2</li>
            <li class="ui-state-default">Item 3</li>
            <li class="ui-state-default">Item 4</li>
            <li class="ui-state-default">Item 5</li>
            <li class="ui-state-default">Item 6</li>
            <li class="ui-state-default">Item 7</li>
        </ul>  
        <script>
        $(function () {
            $("#sortable").sortable({
            placeholder: "ui-sortable-placeholder"
            });
        });
        >/script>

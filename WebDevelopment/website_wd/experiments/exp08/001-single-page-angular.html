In this experiment, I made a single page application using AngularJS which I will use in my project. Sites that mimic the single page app behavior are able to provide the feel of a phone/tablet application. Angular helps to create applications like this easily. Code snippet below. 

    
    <header>
        <nav class="navbar navbar-default">
        <div class="container">
        <div class="navbar-header">
        <ul class="nav navbar-nav navbar-right">
        <li><a href="#"><i class="fa fa-home"></i> Home</a></li>
        <li><a href="#about"><i class="fa fa-shield"></i> About</a></li>
        <li><a href="#contact"><i class="fa fa-comment"></i> Contact</a></li>
                    </ul>
                </div>
            
            </div>
        </nav>
    </header>
    <!-- MAIN CONTENT AND INJECTED VIEWS -->
        <div id="main">
        <!-- angular templating -->
        <!-- this is where content will be injected -->
        <div ng-view></div>
    </div>

The Bootstrap Rating Input with custom icon classes

My rating:

In this experiment, I worked on the Bootstrap Rating input plugin. When you click on the a rating, the value is alerted. I'll use this plugin for rating movies in my project. Code snippet below. 

    <p>My rating: <input type="number" name="your_awesome_parameter" id="some_id"
             class="rating" data-clearable="remove" data-icon-lib="fa"
             data-active-icon="fa-heart" data-inactive-icon="fa-heart-o"
             data-clearable-icon="fa-trash-o" /></p>
    
    <script>
        $(function () {
         $('input').on('change', function () {
           alert("Changed: " + $(this).val())
         });
      });
    </script>

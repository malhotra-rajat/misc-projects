Home Page



The Home page provides the following data from the Rotten Tomatoes API

  • Upcoming movies: The Home Screen shows the upcoming movies in a beautiful poster grid
  • Movies in theateres: The Home Screen also shows the movies already in theatres in a table with the following data:
    • Movie Poster
    • Title
    • Rating
    • Release Date
    • Critics Score
    • Audience Score

The Movies in theatres list is paginated. There are "Next" and "Previous" buttons below the table as shown in the screenshot below.

Future Scope

I thoroughly enjoyed developing Moviebook and would definitely add more features to the project in the future. Some features I would like to add are:

  • Searching for user profiles
  • Link a movie if it is available on Netflix/Amazon Instant Video/Hulu Plus
  • Link to a ticket booking website
  • Upgrade to AngularJS 2.0 when it comes out. It has got good early reviews.

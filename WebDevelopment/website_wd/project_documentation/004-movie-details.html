Movie Details Page



The Movie Details page provides the following data from the Rotten Tomatoes API

  • Movie Poster
  • Title
  • Studio
  • Synopsis
  • Release Date
  • Critics Score
  • Audience Score
  • Starring
  • Directed By

If the user is not logged in, he cannot like or comment on a movie and the page is shown as in the screenshot above.

If the user is logged in, he can like or comment on a movie and the page is shown as in the screenshot below.
